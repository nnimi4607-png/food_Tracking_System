🍽️ Food Tracking System

A modern and responsive Food Tracking System Web Application that helps users track their daily food intake, monitor calories, and maintain a healthy lifestyle.

🚀 Features
🔐 User Authentication (Signup/Login)
🍔 Add & Track Meals (Breakfast, Lunch, Dinner, Snacks)
📊 Calorie & Nutrition Tracking (Protein, Carbs, Fats)
📈 Dashboard with Charts & Progress Indicators
📅 Food History with Date Filter
🌙 Dark/Light Mode (Optional)
📱 Fully Responsive Design
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Python (Flask/Django)
Database: SQLite / MySQL
API: Nutrition API (for food data)
📂 Project Structure
food-tracking-system/
│── static/        # CSS, JS, Images
│── templates/     # HTML files
│── app.py         # Main backend file
│── database.db    # Database
│── requirements.txt
│── README.md
⚙️ Installation & Setup
Clone the repository:
git clone https://github.com/your-username/food-tracking-system.git
Navigate to project folder:
cd food-tracking-system
Install dependencies:
pip install -r requirements.txt
Run the application:
python app.py
Open in browser:
http://127.0.0.1:5000/
